# JavaScript入門1

## 変数2

今回も変数について学んでいきます

おさらいです

```javascript
    // 変数valueに文字列"value"が格納された
    var value = "value"
    console.log(value) 
    // "value"が出力される
```

ここではキーワード`var`を使用して変数を定義しましたが、

変数を定義するキーワードは他にもあります。

`let`と`const`です

これら3つにはそれぞれ違いがあります

### `var`

- 再宣言可能
- 再代入可能

### `let`

- 再宣言不可
- 再代入可能

### `const`

- 再宣言不可
- 再代入不可

今は難しいと思いますので、次のステップから順をおって説明していきます。
